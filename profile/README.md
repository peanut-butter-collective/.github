## PBC staff: set up your workstation

Your work happens in a `west`-managed set of repositories. One command sets up a fresh machine: it installs the tooling, signs you in to GitHub, and materialises every repository into a local workspace.

**macOS / Linux**
```sh
curl -fsSL https://raw.githubusercontent.com/peanut-butter-collective/.github/main/install.sh | sh
```

**Windows (PowerShell)** (first enable Developer Mode: Settings → System → For developers):
```powershell
irm https://raw.githubusercontent.com/peanut-butter-collective/.github/main/install.ps1 | iex
```

You'll be asked to sign in with the GitHub CLI; you need to be a member of this organisation to clone the repositories.
